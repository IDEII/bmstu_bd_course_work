# Курсовая работа по курсу "Базы данных" в МГТУ
## Описание проекта
Веб-приложение, созданное в рамках курсовой работы на Golang для регистрации участников на турниры. Приложение предоставляет функциональные возможности регистрации пользователей, организаций, позволяет организовывать турниры в том числе отслеживать и принимать/откланять заявки участников (пользователей) на турниры как в качестве непосредственных участников, так и тренеров.

## Функциональные возможности
### Аутентификация и регистрация
* Регистрация пользователей
* Аутентификация зарегестрированных пользователей
* Хранение данных пользователей в базе данных
### Личный кабинет участника
* Просмотр информации о пользователе/организации
* История участия в турнирах
* Регистрация на турниры
### Личный кабинет организации
* Создание клубов и менеджент участников
* Организация турниров
* Проведение жеребьевки
* создание сетки турнира по различным схемам
### Общие возможности
* Просмотр неконфеденциальных данных всех пользователей
* Просмотр турниров
## Технические особенности
### Используемые технологии
* GUI: Angular
* База данных: Microsoft SQL Server
